## COD Warzone Daily Stats
Discord bot for calculating daily player stats for Call of Duty Warzone.

### Instructions
- Discord bot command: `!cds <psn|atvi> <username>`
- Platforms:
 1 - `psn`: PlayStation Network
 2 - `atvi`: Activision ID
